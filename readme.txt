SnapThat adds a floating feedback button to your site to capture annotated screenshots and create tasks with comments, sent to your snapThat.dev workspace.

== Description ==
Use a small on-page widget to log issues with annotated screenshots, create tasks, and collaborate with your team. Configure brand color, pin color, position, theme, and visibility (by roles or public).

== Installation ==
1. Upload the plugin folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to *Settings → SnapThat Plugin* and configure options.
